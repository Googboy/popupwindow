今天做的是popupwindow（悬浮框），他和alertdialog的区别就是：alertdialog不能指定显示位置，只能默认在屏幕中间，（当然也可以通过设置WindowManager参数来改变位置）
而popupwindow是可以指定显示位置的，随便哪个位置都有可以，更加灵活。
其中遇到的第一个错误就是：代码运行模拟器并没有显示悬浮框，查看了几遍才发现由于忘记添加显示popupwindow的代码了 - -!
添加以后就可以正常显示了，其中一共有3个标签，点击每一个都可以在界面上显示你所点击的内容。
还有就是viewpager（页面切换），设置3个界面，左右滑动展现不同的页面，后来冒出个想法又自定义一个跳转界面，点击按钮（可以设置为登陆界面），跳转到viewpager。
